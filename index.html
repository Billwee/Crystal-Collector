Crystal Collector

Welcome to my Crystal Collector game! In this game you must try to match the target by clicking on the four crystals below to add points to your score. The game begins by showing a target number you need to reach with your score. You add points to this score by clicking the four crystals below. Each crystal as a random value assigned to it that is added to your score when they are clicked. If you match the target score you win, but if you go over the target you lose. Then a new round will start. When that happens each crystal is assigned a new random value and a new target value is displayed. Good Luck!
